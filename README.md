# RvThereYet-Rammstein-Music
Adding Rammstein Music to RVTherYet

Installation:
Drag and drop all files from the Eng\Ride\Content\Paks folder into the following path: "Steam\steamapps\common\Ride\Ride\Content\Paks"

Note:
The game automatically cuts music tracks short and does not support full-length custom cassettes. As a result, all new custom cassettes are truncated to fit the game's timing constraints.

Установка:
Перетащите все файлы из папки Rus\Ride\Content\Paks по следующему пути: "Steam\steamapps\common\Ride\Ride\Content\Paks".

Примечание:
Игра автоматически сокращает музыкальные треки и не поддерживает полноформатные пользовательские кассеты. В результате все новые пользовательские кассеты обрезаются в соответствии с временными ограничениями игры.
